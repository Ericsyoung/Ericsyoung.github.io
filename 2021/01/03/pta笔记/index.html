3. 第3章-17 删除字符

## 例1
strs = input().strip()
tar = input().lower().strip()
for item in strs.lower():
    if item == tar:
       strs = strs.replace(item,'')
print('result: {}'.format(strs))

## 例2
lists= [1,2,3,4]
for item in lists:
    item = 0

做这道题时想了一下哪些情况可以动态改变列表，其实这就是按值传递和按引用传递的问题了。例1中replace函数是可以实现list在循环中动态更改的。在例2中，0仅仅是被赋给了item变量，无法传递到lists中，要想动态改变lists中的值，需要使用索引的方式，即lists[i] = 0这样的写法。replace函数其实就是实现了对strs进行了按引用传递。

按值传递重要特点：传递的是值的拷贝，也就是说传递后就互不相关了。

按引用传递指的是在方法调用时，传递的参数是按引用进行传递，其实传递的引用的地址，也就是变量所对应的内存空间的地址。

4. 第4章-4 验证“哥德巴赫猜想”

n = int(input())
q = 2
for num in range(2,n-1):
    flag = 0
    for item in range(2,num): # num是否是素数，用于赋值q
        if num % item == 0:
            flag = 1
            break
    for elem in range(2,n-num): # n-num，即p是否是素数
        if (n-num) % elem == 0:
            flag = 1
            break
        if q <= num and flag == 0:
            q = num
print('{} = {} + {}'.format(n,n-q,q))

最原始的写法，O(n^2)，输入过大时会运行超时。

本来想直接读出所有素数，然后遍历，直到两个素数都在list里，然后我的好友sstj来教我做事了：

def isPrime(n):
    # Corner Cases
    if (n <= 1):
        return False
    elif (n <= 3):
        return True
    elif (n % 2 == 0 or n % 3 == 0):
        return False
    i = 5
    while i * i <= n:
        if (n % i == 0 or n % (i + 2) == 0):
            return False
        i += 6

    return True
    
def main():
    start = time.time()
    n = int(input())
    res = None
    for q in range(2, n - 1):
        if isPrime(q) and isPrime(n - q):
            res = (q, n - q)
            break
    if res == None:
        print("Not found")
    else:
        print('{} = {} + {}'.format(n, q, n - q))
        print("finished finding in {}".format(time.time() - start))
    return

main()

5.第4章-6 输出前 n 个Fibonacci数

“如果使用循环，程序的性能可能会更高，如果使用递归，程序可能更容易理解，如何选择要看什么对你来说更重要。”

为什么递归耗时更长：

以斐波那契数列举个例子理解下为什么会超时

def fib1(n: int):
	# 最基础的实现方法
    if n <= 2:
        return 1
    return fib1(n - 1) + fib1(n - 2)

我们都知道斐波那契数列从第三项开始每一项为前两项的和，那么如果我们要求第七项，那么流程如下：

从图中可以看到，重复计算了两遍5，三遍5，四遍3。如果项数很大，重复计算的次数只会越来越多

优化方法：

  1. 缓存每次新计算的结果，不要一上来，就申请O(n)的空间。

    from functools import lru_cache
@lru_cache(10 ** 8)
def fibo(n: int):
    if n <= 2:
        return 1
    return fib3(n - 1) + fib3(n - 2)

  2. 使用一个数组，存放曾经计算过的结果，下次遇到同样的计算，直接获取结果

    def fibo(n: int):
    if n <= 2:
        return 1
    memo = [-1] * n
    memo[0] = memo[1] = 1
    for i in range(2, n):
        memo[i] = memo[i - 1] + memo[i - 2]
    return memo[n - 1]

    当然个人更喜欢下标从1开始的形式

    def fibo(n):
    if n <= 2:
        return 1
    memo = [-1] * (n+1)
    memo[1] = memo[2] = 1
    for i in range(3, n+1):
        memo[i] = memo[i-1] + memo[i-2]
    return memo[n]
